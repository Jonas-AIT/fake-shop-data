API für Statusabfragen von Fake Shops

download data

Das Programm dient dazu, die Fake Shops, die auf der Watchlist Internet gelistet sind auf ihren Status zu überprüfen. Es führt alle Tests vollautomatisch in einer konfigurierten Frequenz durch. Die werden in einer zentralen SQLite Datenbank gespeichert. Die Datei kann sich mit dem Feld "download data" heruntergeladen werden

Wenn Sie eine spezielle URL prüfen wollen, geben Sie diese in das Eingabefeld neben ein. Sie werden anschließend auf eine Seite mit den Ergebnissen weitergeleitet. Da der Test Zeit in Anspruch nehmen kann, haben Sie bitte Geduld.
